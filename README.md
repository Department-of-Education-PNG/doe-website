# Department of Education, Papua New Guinea - Website

This repository contains the source code for the official website of the Department of Education, Papua New Guinea. Built as a modernized, responsive Single Page Application (SPA) using HTML, CSS, and Vanilla JavaScript.

## 📌 Project Overview
The goal of this project is to provide a premium, accessible, and user-friendly portal for students, parents, teachers, and the general public to access educational resources, news, and services.

## 🚀 Technologies Used
- **Frontend Core:** HTML5, CSS3, Vanilla JavaScript
- **Icons & Assets:** FontAwesome, Modern SVG icons
- **Architecture:** Component-based SPA routing without a heavy framework
